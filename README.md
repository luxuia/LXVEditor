# LXVEditor

基于ReoGrid的 CSV扩展编辑器， 给csv提供更多扩展

- [x] 自定义的分隔符       ---sep:∤
- [x] 多sheet支持         ---sheet:xxx
- [ ] 注释                ---meta:marco:xxx
- [ ] 图片等二进制数据     ---meta:tex:base64

导出的数据大致如：
```
---sep:∤
---sheet:sheet1
col1∤col2∤col3
v1∤v2∤v3
---sheet:sheet2
col1∤col2∤col3
v1∤v2∤v3
```
